C-Mini language reference

Types

int — 64-bit integer. Only type supported.

Operators

+ - * arithmetic · == != < <= > >= comparison. No division.

Statements

  • int x = expr; — declaration
  • x = expr; — assignment
  • if (cond) { … } else { … }
  • while (cond) { … }
  • return expr; — result in X00

Registers

Variables: x1–x15 · Temps: x16–x25 · Return: x0

What's supported

Compiled with:

gcc -O1 -std=c17 -Wall -Wextra

Full standard library — stdio.h, stdlib.h, string.h, math.h, etc. All operators including / and %. Program stdout and stderr are captured and displayed.

Tip

Programs that run longer than the timeout are killed automatically.
